#  AI Knowledge System (Second Brain)

AI Knowledge System is a full-stack **Second Brain** application built using **Next.js**.  
It allows users to capture ideas, notes, and insights, store them in a database, and enhance them using AI-powered features like summarization and auto-tagging.

The project focuses on **clean UI, solid architecture, and real-world full-stack practices**.
